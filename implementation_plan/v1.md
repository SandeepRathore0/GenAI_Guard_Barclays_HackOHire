### 2. Model Training Pipeline
#### [NEW] `train_models.py`(file:///d:/Project/GenAi-Guard/GenAi-Guard/train_models.py)
We need to populate the `models/` directories for each module so that the `.predict()` methods work. I will write a script `train_models.py` that will use `scikit-learn` to build lightweight baseline models.
- **`text_guard/models/phishing_clf.pkl`**: A text classification pipeline (`TfidfVectorizer` + `LogisticRegression`) trained on a small synthetic dataset of phishing and safe emails.
- **`text_guard/models/injection_clf.pkl`**: A text classification pipeline trained to detect prompt injection phrases (e.g., "ignore previous instructions").
- **`web_guard/models/url_phishing_clf.pkl`**: A text classification pipeline trained on safe and malicious URLs.
- **`file_guard/models/malware_sig_model.pkl`**: A text classification pipeline that takes a filename and detects suspicious extensions (like `.pdf.exe`).
- **`audio_guard/models/voice_auth_model.pkl`**: A simple dummy model that can accept a string (like `"dummy_audio_features"`) and output a prediction (0 or 1).
