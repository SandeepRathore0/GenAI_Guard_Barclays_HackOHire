# GenAI Guard Setup & Integration Plan (v4)

## Goal Description
The user wants to train robust models for **Audio Guard** and **File Guard**, similar to what was done for Text and Web. Furthermore, the user wants the **Sandbox Integration** to act as an isolated environment. When a file or audio is uploaded, it should be analyzed within the Sandbox so that any malicious executable or malformed file won't break the main FastAPI system.

## Proposed Changes

### 1. Robust Audio & File Model Training
We will create a script `train_media_models.py` to train robust ML models for Audio and Files.
#### [NEW] `train_media_models.py`(file:///d:/Project/GenAi-Guard/GenAi-Guard/train_media_models.py)
- **Audio Deepfake Model**: Use `librosa` to extract MFCC (Mel-frequency cepstral coefficients) features from audio files, and train a Random Forest classifier to distinguish between human voice and AI-generated deepfakes. 
- **Malware File Model**: Extract structural features from files (e.g., Shannon entropy, byte distribution, file size) and train an XGBoost/Random Forest classifier to detect anomalies typical of malware/pharming.

### 2. Implement the Sandbox API Environment
The sandbox is meant to be a separate Flask application running on port 5000 acting as our isolated detonation/analysis chamber.
#### [MODIFY] `sandbox/app.py`(file:///d:/Project/GenAi-Guard/GenAi-Guard/sandbox/app.py)
- Build a Flask API with an endpoint `/analyze_media`.
- This endpoint will receive files from the main app, run the robust Audio or File models *inside* the sandbox, and return the secure analysis report back to the main app.

### 3. Forward File/Audio Requests to Sandbox
We need to modify the main backend to stop analyzing files locally and instead proxy them to the sandbox.
#### [MODIFY] `app/modules/file_guard/routes.py`(file:///d:/Project/GenAi-Guard/GenAi-Guard/app/modules/file_guard/routes.py)
- Update the `/scan-file` endpoint to forward the uploaded file bytes to `http://localhost:5000/analyze_media`.
#### [MODIFY] `app/modules/audio_guard/routes.py`(file:///d:/Project/GenAi-Guard/GenAi-Guard/app/modules/audio_guard/routes.py)
