# GenAI Guard Setup & Integration Plan (v5)

## Goal Description
The user wants to transition from just having isolated "scanners" to demonstrating the core value of the product: an **Integrated Guard Interceptor**. We will build a functional chatbot interface and wrap it with a helper function (the "Guard") that intercepts prompts, scans them for prompt injections or malicious intent *before* they reach the bot, and flags/blocks them.

## Proposed Changes

### 1. Build the Secure Chatbot Backend Route
We will create a new endpoint in the FastAPI application that acts as the proxy/interceptor.
#### [NEW] `app/modules/secure_chat/routes.py`(file:///d:/Project/GenAi-Guard/GenAi-Guard/app/modules/secure_chat/routes.py)
- Create an endpoint `/chat`.
- **The Helper Function (Guard)**: When a prompt is received, it will first call the underlying logic from `text_guard` (our XGBoost model + HuggingFace fallback) to check for `PROMPT_INJECTION` or `PHISHING`.
- If blocked: Return a security error to the user without calling the LLM.
- If safe: Call a mock "LLM Responder" logic (or pass through to a real API if available) and return the AI response.

### 2. Update Main Application
#### [MODIFY] `app/main.py`(file:///d:/Project/GenAi-Guard/GenAi-Guard/app/main.py)
- Import and include the new `secure_chat` router under the prefix `/api/v1/secure-chat`.

### 3. Create the "Protected Chat" UI
We will add a dedicated interface in the dashboard to showcase this live interception.
#### [MODIFY] `app/simple_dashboard/index.html`(file:///d:/Project/GenAi-Guard/GenAi-Guard/app/simple_dashboard/index.html)
- Add a new "Protected AI Chat" tab to the Quick Launch menu.
- Build a chat window UI (similar to ChatGPT) where the user types messages.

#### [MODIFY] `app/simple_dashboard/script.js`(file:///d:/Project/GenAi-Guard/GenAi-Guard/app/simple_dashboard/script.js)
- Add the frontend Javascript to handle the chat state.
- If the server returns a block response, display a red `[BLOCKED BY GENAI GUARD]` message in the chat feed to visually demonstrate the security layer in action.
